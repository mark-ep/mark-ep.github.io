That was graceful wasn't it – the way I just slipped those in there. With just as much grace and natural flow, here's a big block of code:

def mandlebrot(x, y, max_iters=1000):
  c = complex(x, y)
  z = complex(0, 0)
  
  for i in range(max_iters):
    z = z * z + c
    if abs(z) >= 2:
      return max_iters
  return False
